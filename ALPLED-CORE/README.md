# AIDLC CORE

AIDLC CORE는 AI-DLC 기반 의료기기 RA 문서 자동화 플랫폼의 FastAPI/RAG/Agent 백엔드입니다. 기존 ALPLED의 요구사항 추출, 산출물 생성, 승인 검토, RAG 검색, 워크플로우 구조를 유지하면서 의료기기 개발 및 RA 문서 작성 보조 도메인에 맞게 설명과 역할을 재정의했습니다.

> 본 프로젝트는 의료 진단 또는 치료 목적의 소프트웨어가 아니라, 의료기기 개발 및 RA 문서 작성 과정을 보조하기 위한 포트폴리오용 문서 자동화 플랫폼입니다.

## 백엔드 역할

- 의료기기 개발 자료 기반 RA 문서 초안 생성
- 요구사항과 시험 시나리오 자동 매핑
- 위험요소와 검증항목 연결
- 누락 문서 및 문서 간 불일치 검출
- 근거 기반 RAG 문서 생성
- 규제 문서 템플릿 기반 출력
- 승인 요청 문서의 의미적 정합성 검토

## 생성 대상 문서

- 요구사항정의서
- 소프트웨어 요구사항 명세서
- 위험관리표
- 검증 및 밸리데이션 계획서
- 통합시험 시나리오
- 변경관리 기록
- 추적성 매트릭스

## 주요 모듈

| 경로 | RA 플랫폼 관점 역할 |
| --- | --- |
| api/ | RA 문서 생성 및 검토 API 엔드포인트 |
| agents/ | 요구사항, 위험관리, 검증문서, 승인 검토 에이전트 |
| rag/ | 규제 근거와 내부 산출물 검색 기반 RAG |
| templates/ | RA 문서 템플릿 및 출력 구조 |
| workflow/ | 문서 생성, 검증, export 워크플로우 |
| schemas/ | 문서 JSON 구조와 검증 스키마 |
| data_pipeline/ | 규제 문서와 참고 자료 chunking/ingest 파이프라인 |

## 범위 제한

AIDLC CORE는 의료영상 판독, 질병 진단, 처방 추천, 치료 계획 수립을 수행하지 않습니다. 입력 자료와 규제 근거를 바탕으로 RA 문서 작성과 문서 간 정합성 검토를 보조하는 백엔드입니다.
