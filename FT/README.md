# AIDLC FT

AIDLC FT는 AI-DLC 기반 의료기기 RA 문서 자동화 플랫폼에서 요구사항 정규화와 문서 구조화 성능을 개선하기 위한 파인튜닝 실험 영역입니다. 기존 RFP 요구사항 생성 실험을 의료기기 RA 문서 작성 보조 관점으로 확장하여, 제품 설명과 개발 자료에서 검증 가능한 요구사항과 추적 가능한 문서 구조를 생성하는 모델 실험으로 정리합니다.

> 본 프로젝트는 의료 진단 또는 치료 목적의 소프트웨어가 아니라, 의료기기 개발 및 RA 문서 작성 과정을 보조하기 위한 포트폴리오용 문서 자동화 플랫폼입니다.

## 목표

- 장문의 개발 자료에서 RA 요구사항 후보 추출
- 중복·부분중복 요구사항 병합
- 요구사항 ID, 근거, 검증 기준, 관련 위험 항목 구조화
- 요구사항정의서와 소프트웨어 요구사항 명세서 초안 품질 개선
- 추적성 매트릭스 생성을 위한 요구사항 단위 정규화

## RA 문서 자동화 연결

FT 결과는 다음 문서 생성 단계에 활용되는 것을 목표로 합니다.

- 요구사항정의서
- 소프트웨어 요구사항 명세서
- 위험관리표의 관련 요구사항 연결
- 통합시험 시나리오의 요구사항 매핑
- 추적성 매트릭스의 요구사항 ID 정규화

## 비의료 판단 범위

이 파인튜닝 실험은 의료 판단, 진단, 처방, 치료 추천을 학습하지 않습니다. 문서 작성 보조, 요구사항 구조화, 문서 간 추적성 확보를 위한 언어 모델 실험입니다.
