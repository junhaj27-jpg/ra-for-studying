추적성/위험관리/시험 시나리오 연결 구조

AIDLC-RA는 의료기기 실제 인허가를 대신하지 않고, RA 문서 작성 보조와 문서 정합성 검토 지원 관점에서 산출물 간 연결을 보여줍니다.

요구사항 -> 위험관리

사용 목적과 기능 요구사항을 위험관리표의 위해요인, 위해상황, 위험통제 항목과 연결합니다.

위험통제 -> 검증시험

위험통제 방법이 통합시험 시나리오와 검증 및 밸리데이션 계획서에서 확인되는지 검토합니다.

변경관리 -> 재검증

요구사항, 알고리즘, UI, 문구 변경이 추적성 매트릭스와 재검증 필요 여부에 반영되는지 확인합니다.

요구사항-위험-시험 연결 그래프

서비스 모듈은 추적성 매트릭스 데이터를 노드와 엣지로 변환하고, 화면은 RA 검토자가 연결 단절을 빠르게 볼 수 있도록 표현합니다.
