AIDLC-RA: 의료기기 개발문서를
RA 제출 준비 흐름으로 구조화합니다

의료기기 개발자가 인허가 및 품질문서 작성 과정에서 요구사항, 위험관리, 검증시험, 추적성 매트릭스를 일관되게 작성할 수 있도록 지원하는 RA 문서 자동화 플랫폼입니다.

본 프로젝트는 의료 진단 또는 치료 목적의 소프트웨어가 아니라, 의료기기 개발 및 RA 문서 작성 과정을 보조하기 위한 포트폴리오용 문서 자동화 플랫폼입니다.

Input

개발 근거 자료

제품 개요, 요구사항, 회의록, 시험 자료, 변경 이력

Generate

RA 문서 생성

내부 가이드와 표준 양식 기반 초안 생성

Review

정합성 검토

누락 문서, 불일치, 미검증 항목 확인

AI는 의료 판단을 수행하지 않으며, 생성 에이전트와 검증 에이전트가 RA 문서 작성 보조와 문서 정합성 검토 지원 역할만 수행합니다.

RA 문서화 업무 흐름

의료기기 개발 생명주기 문서화를 기준으로 업로드, RAG 검색, 문서 생성, 정합성 검토, 버전 확정 흐름을 구성합니다.

1

의료기기 개발 자료 업로드

제품 설명, 사용자 요구사항, 회의록, 시험 자료, 변경 이력, 규제 근거 문서를 등록합니다.

2

근거 기반 RAG 문서 생성

내부 가이드, 표준 양식, 프로젝트 자료를 참고해 요구사항정의서와 SRS 초안을 생성합니다.

3

요구사항-위험-시험 자동 매핑

요구사항 ID, 위험 ID, 시험 ID를 연결해 추적성 매트릭스의 기본 구조를 만듭니다.

4

RA 문서 정합성 검토

누락 문서, 사용 목적 불일치, 위험통제 미검증, 변경 영향 누락을 검토합니다.

5

RA 문서 패키지 이력 관리

검토 완료 문서를 버전으로 확정하고 변경관리 기록과 연결합니다.

산출 문서와 연결 구조

RA 포트폴리오에서 중요한 요구사항, 위험관리, 검증시험, 추적성 연결을 화면에서 확인할 수 있도록 구성합니다.
